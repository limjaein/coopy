# Coopy!

**Coopy**는 **음식 레시피 공유 프로그램**입니다.
재료와 가격 등의 세부조건을 통해 마음에 드는 레시피를 찾을 수 있고, 자신만의 레시피를 공유할 수 있습니다.

## 기술 스택

- Spring Boot
- JPA
- Kotlin
- MySQL
